### 1. Prerequisites

### 3. Install Dependencies
```bash

```

### 4. Configure Environment
Create a `.env` file in the `002_Task` root directory and add your database credentials:
```env
DB_USER=your_postgres_user
DB_PASSWORD=your_postgres_password
DB_HOST=localhost
DB_PORT=5432
DB_NAME=your_database_name
PORT=3000
```

### 5. Run the Application
```bash
npm start
```
